### PhaseForensics 프로젝트 현황 파악 완료

**프로젝트 요약:** DT-CWT 위상 일관성을 이용한 Training-Free 합성 이미지 탐지 시스템. CVPR 2027 / ECCV 2026 목표.

**핵심 아이디어:**
- 기존 방법: FFT/DCT 진폭(amplitude)만 사용
- 본 연구: 위상(phase) 일관성 위반을 최초로 활용
- 실제 카메라 PSF → 스케일 간 위상 정렬 / 생성 모델 디코더 → 위상 비일관성
- JPEG 압축에 강건 (Oppenheim & Lim 1981)

**준비 완료 항목:**

| 항목 | 상태 |
|------|------|
| 아이디어 검증 (50K 논문 분석, #1 선정) | ✅ |
| 3인 전문가 리뷰 (33~43/50점) | ✅ |
| 이론 문서 4편 + 관련 연구 정리 | ✅ |
| 코어 코드 5개 모듈 (~600 LOC) | ✅ |
| 파일럿 스터디 스크립트 | ✅ |
| 8주 실험 로드맵 | ✅ |
| 리스크 분석 + GO/NO-GO 기준 | ✅ |

**코드 구조:**
```
src/
├── phase_extraction.py      # DT-CWT 위상 추출 (PhaseExtractor)
├── coherence_metrics.py     # ISPC + CDPGC 메트릭 계산
├── statistical_detection.py # GGD 피팅 + Neyman-Pearson 검출
└── detector.py              # 통합 오케스트레이터 (PhaseForensicsDetector)

scripts/pilot_study.py       # Week 1 GO/NO-GO 판정 스크립트
```

**미구현 항목:**
- ❌ 유닛 테스트 (tests/ 비어있음)
- ❌ 실험 결과 없음 (results/ 비어있음)
- ❌ 시각화 스크립트
- ❌ 압축 벤치마크 자동화
- ❌ Ablation 프레임워크

**즉시 다음 단계: Week 1 파일럿 스터디**
- 9개 생성기별 100장 → ISPC/CDPGC 히스토그램 + Cohen's d
- GO: ≥7/9 생성기에서 d > 0.5
- CONDITIONAL: 5~6개 (amplitude+phase 융합)
- NO-GO: <5개 (FoMFP/FAP로 전환)

### 6개 검수 에이전트 결과 종합

| 심각도 | 건수 | 핵심 내용 |
|--------|------|-----------|
| CRITICAL | 5 | ISPC/CDPGC 수학적 오류, config 미연결, 입력 검증 없음 |
| HIGH | 8 | GGD 피팅 오류, NP 검출기 미구현, 임계값 하드코딩 |
| MEDIUM | 8 | 성능 최적화, gradient_method 미구현, 직렬화 보안 |
| LOW | 6 | 타입힌트, 디버그 로깅, 미사용 import |

**핵심 CRITICAL 이슈 3가지:**
1. ISPC: spatial axis에 대해 circular mean → 원래는 scale pairs 축으로 해야 함
2. CDPGC: gradient magnitude 사용 → 원래는 방향 projection + 공간적 variance map 필요
3. GGD: 비음수 데이터에 floc=0 대칭 분포 피팅 → 부적절

**데이터셋 현황:**
- ✅ FFHQ 70,000장 (real), AI_Generated 7개 생성기 x 1,000장 (fake)
- ❌ COCO, ImageNet, GenImage, CNNDetection 미존재
- → 파일럿 스터디는 FFHQ + AI_Generated로 즉시 가능

**환경:** ✅ 모든 의존성 설치 완료, 스모크 테스트 통과

이후 5개 에이전트를 병렬 투입하여 모든 이슈를 수정:

| 작업 | 상태 | 내용 |
|------|------|------|
| 수학적 오류 수정 | ✅ | ISPC(scale-pair 축 누적), CDPGC(방향 projection + x/y축 수정), GGD(중심화), NP(GGD 우도비) |
| 엔지니어링 개선 | ✅ | config.py, 입력 검증, from_config, JSON 직렬화, pyproject.toml |
| 파일럿 스터디 완성 | ✅ | 실제 데이터 경로, 히스토그램 시각화, JPEG 압축 테스트, tqdm, argparse, 이메일 알림 |
| P0 테스트 38개 | ✅ | 4개 모듈 전체 커버, 모두 통과 |
| config 보강 | ✅ | 실제 데이터셋 경로, 파일럿 스터디 설정, 이메일 설정 |
